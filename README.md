# Sushi Order
## Get Code
1. go to work folder "~/"
2. git clone https://github.com/songqing2022/sushi.git

## Build code
1. enter project home folder "/sushi"
2. run maven command: mvn clean package

## Test code
1. enter project home folder "/sushi"
2. run command: mvnw test

## Run app
   run java command: java -jar ./target/sushi-1.0-SNAPSHOT.jar
